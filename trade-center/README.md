# Trade Command Center — מערכת ניהול טריידים

תיקייה עצמאית שמצטרפת לפרויקט **wallstreet-view** לצד ה־Momentum Desk הקיים, בלי לשנות בו דבר.

## מה יש כאן

| קובץ | תפקיד |
|---|---|
| `index.html` | הכלי המלא — מושל סיכון יומי, תכנון עסקה עם חישוב כמות, פוזיציות, יומן תחקיר, ספר חוקים והגדרות. הכול רץ בדפדפן ונשמר ב־localStorage. |
| `INTEGRATION.md` | הצעת הסנכרון המלאה עם Momentum Desk — מה כבר בוצע ומה מוצע לשלבים הבאים. |

## מסך „עסקה מהירה" — דף הבית של הכלי

הזרימה המלאה בקליק אחד מה־Desk: לוחצים „תכנן" על מניה ← נוחתים במסך המהיר עם הסימול והמחיר החי ← מזינים רק סטופ (או בוחרים צ'יפ 0.5%–2%) ← מקבלים מיד כמות, סיכון דולרי, יחס נטו והחלטה: מאושר / מאושר בזהירות / פסול. מושל הסיכון היומי רץ ברקע. יעד ריק = 2R אוטומטי. „שמור לפוזיציות" — וזהו.

התכנון המלא (סליידרים, ציוני איכות, התנהגות, חפיפות) זמין בלשונית „תכנון מלא" למי שרוצה להעמיק — שני המסכים תמיד מסונכרנים על אותם מספרים.

## מה כבר מסונכרן (בוצע בקוד)

1. **מחירים חיים** — הכלי מתחבר לאותו Cloudflare Worker שכבר משרת את ה־Momentum Desk (`momentum-yahoo-prices.sh6doron.workers.dev`), כולל טרום/אחרי מסחר:
   - במסך „עסקה חדשה": פס „מחיר חי" עם מחיר עדכני, אחוז שינוי ומצב שוק, וכפתור „קבע ככניסה".
   - במסך „פוזיציות": כפתור „רענן מחירים" + מצב אוטומטי (כל 30 שניות). רווח/הפסד, R ופריצת סטופ מתעדכנים מהשוק.
   - התראת **„סטופ נפרץ"** על פוזיציה שהמחיר חצה בה את נקודת הפסילה.
2. **קליטת סימול מה־Desk** — פתיחה עם `?symbol=SOXL` ממלאת את הסימול ועוברת ישר למסך תכנון עסקה.
3. **ערכת נושא משותפת** — אם אין העדפה מקומית, הכלי יורש את מצב בהיר/כהה של ה־Momentum Desk (אותו דומיין ⇒ אותו localStorage).
4. **קישור חזרה** — כפתור „מסך המומנטום" בסרגל העליון.

מעתיקים את התיקייה `trade-center/` כמו שהיא לשורש הריפו. לאחר פרסום ב־GitHub Pages הכלי יהיה זמין בכתובת:

```

```

## מה לא נגענו בו

- `index.html` הראשי (Momentum Desk) — ללא שינוי.
- `worker/worker.js` — ללא שינוי; משתמשים בו כפי שהוא.

הצעדים הבאים (כפתור „תכנן עסקה" בתוך ה־Desk ועוד) מפורטים ב־`INTEGRATION.md` ויבוצעו רק באישורך.
